# Iris Flower Classification using KNN

## Project Overview
This project uses the Iris Dataset and K-Nearest Neighbors (KNN) algorithm to classify iris flowers into three species.

## Technologies Used
- Python
- Scikit-Learn
- Google Colab

- Accuracy: 100%
- Confusion Matrix Generated
